<h1> Vyapaar Crypto Trader </h2>
   <p> A Cryptocurrency Trading Website simulating buying and selling of coins. </p>

<h2> How to Run </h2>
   - npm run dev :    for frontend 
   - nodemon ./backend/stripe.js :   for Stripe
